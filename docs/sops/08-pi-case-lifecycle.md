## 1. Overview

Personal injury (PI) cases follow a different lifecycle than property insurance cases. This SOP covers the full PI case flow from intake through settlement or trial preparation. PI is the firm's secondary practice area.

---

## 2. Intake

1. **Confirm case viability:**
   - ‚úÖ Venue is acceptable
   - ‚úÖ Statute of Limitations (SOL) allows us to proceed
   - ‚úÖ Type of case is one we handle (must involve injuries)
   - ‚úÖ Fee structure discussed

2. **Get client signatures:**
   - ‚úÖ Authority to Represent (contract)
   - ‚úÖ Medical authorizations
   - ‚úÖ Employment authorizations (if applicable)

3. **Conflict check:**
   - If two clients have a possible conflict ‚Üí get a **waiver of conflict**
   - If conflict cannot be waived ‚Üí refer to another attorney with a **fee split contract**

4. **Client intake questionnaire:**
   - Incident details
   - Treatment history
   - Health insurance company info
   - Provider names and contact info

5. **Calculate the lowest applicable SOL ‚Üí calendar it immediately**

‚ö†Ô∏è **SOL calendaring is critical.** Missing a statute of limitations can result in case dismissal and malpractice liability.

## 3. Beginning of Case ‚Äî LOR & Preservation

### 3.1 Send Letters of Representation

| Recipient | Purpose |
|-----------|---------|
| **BI Carrier** (at-fault party's insurer) | Primary claim |
| **Client's own carrier** | PIP, UM, and UIM coverage |
| Request declarations page from client's carrier | Coverage verification |
| **Possible lienholders** | Health insurance, Medicare, etc. |

### 3.2 Preservation Letters

Send preservation letters if needed ‚Äî particularly to businesses with security cameras or other entities that may have evidence.

### 3.3 Records Requests

- Request a copy of the **police report**
- Do an **open records request** if a government agency is involved
- Request **medical and billing records** from all providers
  - Send requests using our own authorizations or via a third-party records service (e.g., Yocierge)
  - ‚Üí See [SOP 11 ‚Äî Medical Records](11-medical-records.md)

### 3.4 Confirm LOA (Letter of Acknowledgment)

- Confirm receipt of an LOA for each LOR sent
- If no LOA received, follow up as needed
- When received, add information from each LOA to the CMS:
  - Adjuster name
  - Subrogation company
  - Claim numbers, etc.

## 4. Pre-Litigation

### 4.1 Investigate

- Search **public records** for defendants (Secretary of State business records)
- Check **social media** ‚Äî look for defendant's Facebook page and posts about the incident
- Save all findings to the case file

### 4.2 Track Treatment

- Confirm treatment status with the client
- Ensure we have names of **all providers** (some clients see specialists later)
- ‚ö†Ô∏è **We want the client to finish treatment prior to sending the demand**

### 4.3 Confirm Receipt of Records

- When client is done treating, confirm all medical records have been received
- If records haven't arrived in **2 weeks**, follow up ‚Äî and keep following up
- If client gets additional treatment, **update the records**

### 4.4 Settlement Authority

- Obtain settlement authority amount from the client
- **Document this:** date, time, amount authorized

## 5. Demand Preparation

1. Prepare and send the demand (‚Üí see [SOP 03](03-demand-drafting-and-service.md) for general demand procedures ‚Äî adapt for PI specifics)
2. Set deadline and calendar it
3. If no response before deadline ‚Üí follow up once in writing
4. Negotiate with the carrier if they reply with an offer

## 6. Beginning of Litigation

If negotiations fail or liability is denied:

1. **Contact client** and explain next steps ‚Äî get approval to litigate
   - ‚ö†Ô∏è Attorneys' fees usually increase when suit is filed ‚Äî need client permission
2. **Prepare suit and discovery requests**
3. **File suit and discovery requests**
4. Add case info to CMS:
   - Case number
   - County
   - Judge
   - etc.
5. Save case expenses to file
6. **Send courtesy copy** of filed suit to adverse insurance companies

## 7. Discovery Phase

### 7.1 Service Follow-Up

| Timeframe | Action |
|-----------|--------|
| 20 days after filing | Check if parties are served |
| If not served | Follow up; serve again as needed |
| 30 days after service | Check if answer has been filed |
| Answer filed | Add counsel name/info to CMS |
| No answer | Contact defendant carriers |
| Still no answer | Move for default judgment |

### 7.2 Defendant's Discovery

- If we receive discovery requests ‚Üí save and calendar due date (‚Üí see [SOP 05](05-discovery.md))
- Complete our discovery responses and serve on opposing parties

### 7.3 Our Discovery

- Follow up in **30 days** to confirm defendants replied
- If no reply ‚Üí send letter of reminder
- If **30+ days late** ‚Üí file Motion to Compel

### 7.4 Depositions

- Determine if OC wants to depose our client ‚Üí schedule and prep client
- Determine whether to depose defendant
- Ensure deposition is paid for and added to expenses

### 7.5 Expert Witnesses

- Determine whether experts are needed
- If so, consult expert and get a report
- Provide report to defendants as part of discovery

## 8. Settlement During Litigation

Once discovery is complete:

1. **Confirm medical records are up to date**
2. **Add related costs** to case expenses
3. **Contact lienholders** and request updated liens
   - Document liens at face value
4. **Contact PIP carrier** to check remaining PIP balance
5. **Contact opposing parties** about mediation
   - Some venues require mediation ‚Äî determine if ours does
   - If parties won't mediate ‚Üí move for a pretrial/trial date
   - ‚ö†Ô∏è **Calendar any trial deadlines ‚Äî VERY VERY IMPORTANT**
   - Trial orders usually contain a mediation requirement

### 8.1 Mediation Preparation

1. Find an agreeable mediator
2. Contact all parties to schedule
3. Pay mediator deposit and sign release (if required)
4. Update liens
5. Confirm payment status of medical bills
6. Determine total case expenses
7. Prepare mediation statements and calculations
8. Prepare in-house mediation prep documents
9. Send mediation statement to mediator
10. **Mediate the case**
11. Pay the mediator

If case settles ‚Üí proceed to Settlement Procedures (below)
If case doesn't settle ‚Üí proceed to Trial Prep (‚Üí separate SOP)

## 9. Settlement Procedures

### 9.1 Payment Setup

1. Send payment info to opposing counsel:
   - ‚úÖ W9
   - ‚úÖ Check and mailing instructions
2. Calendar entry for **20 days** to follow up on check ‚Äî repeat as needed

### 9.2 After Check Arrives

1. Deposit into **IOLTA account**
2. **Obtain lien payoffs:**
   - Send requests to lienholders
   - Obtain payoffs in writing with payment instructions
3. **Obtain medical bill payoffs:**
   - Send requests to providers
   - Obtain payoffs in writing with payment instructions
4. **Calculate case expenses** (ensure mediator and deposition costs are included)

### 9.3 Release Review

- Review releases provided by opposing parties
- ‚ö†Ô∏è **We cannot sign releases** ‚Äî make sure there isn't a signature block for us
- ‚ö†Ô∏è Ensure release is **"exclusive of PIP"**

### 9.4 Distribution

1. Prepare **disbursement authorization:**
   - Itemize: expenses, fees, lien data
2. Write checks for client and lienholders
3. Schedule time for client to come in
4. Have client execute:
   - ‚úÖ Release
   - ‚úÖ Disbursement authorization
5. Provide settlement check to client
6. Send payoff checks to lienholders
7. Send executed release to opposing counsel
8. Sign **Agreed Order of Dismissal** and return to OC
9. Confirm checks clear
10. Confirm AOD is filed and entered
11. **Close file**
