## 1. Overview

This SOP covers onboarding a new employee from account creation through their first week of training. The goal is to get them set up with all required tools and familiar with firm procedures as quickly as possible.

---

## 5. Day 1 ‚Äî OneDrive & File Access

### Accept Shared Folder Invitations

1. Check email for shared OneDrive folder invitations
2. Click invitation link and sign in
3. Accept shared folder invitations for: **"Clio"** and **"Back Up"**

### Add OneDrive Shortcuts (Windows)

1. Open File Explorer
2. Sign into OneDrive account
3. Open OneDrive in browser ‚Üí navigate to **Shared** section
4. Locate shared folders ("Clio" and "Back Up")
5. Click folder ‚Üí select **"Add shortcut to OneDrive"**
6. Folders will appear in File Explorer for easy access

### Add OneDrive Shortcuts (Mac)

1. Open Finder
2. Sign into OneDrive via the desktop app (download from Mac App Store if needed)
3. In browser, go to onedrive.live.com ‚Üí sign in ‚Üí click **"Shared"**
4. Locate shared folders ("Clio" and "Back Up")
5. Click each folder ‚Üí select **"Add shortcut to My files"**
6. Folders will sync and appear in Finder under OneDrive

## 7. Role-Specific Responsibilities

### Paralegal Responsibilities

**Pre-Suit Workflow:**
- Service email document download
- Save files when CC'd on emails
- Client intake form processing
- Reduce fee for referral source clients (except Independent Restoration Services)
- Send LOR after notification from intake specialist
- Follow up on LORs weekly until documents are provided
- Confirm client intake sheet is complete; call client if necessary
- Request documents from client, carrier, contractors, and referral sources
- Draft demands (first draft within 5 days of ATR execution)
- Calendar deadlines for demands and complaints
- Weekly referral source updates (calendar every Friday to send to Bart)

**Litigation Workflow:**
- File complaints via e-filing system
- Calendar litigation deadlines
- Answer and serve discovery
- Manage settlement process
